RecipeBox

RecipeBox is a mobile app for organizing and discovering recipes. Save recipes from anywhere, tag them by cuisine or occasion, and generate grocery lists with a tap. Built because I was tired of scrolling through bookmarks to find that one pasta recipe I saved months ago.

The app uses React Native with Expo for cross-platform support and Firebase for real-time sync across devices. The search is fast and fuzzy, so you can find recipes by ingredient, name, or tag without remembering the exact title.

Tech Stack

React Native Expo Firebase TypeScript

How to View

  1. Install Expo Go on your iOS or Android device
  2. Scan the QR code below or tap the link on your phone
  3. The app will load directly in Expo Go
