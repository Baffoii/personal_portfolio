TaskFlow CLI

TaskFlow is a command-line task manager for developers who prefer to stay in the terminal. It supports smart scheduling based on priority and deadlines, recurring tasks, and a clean ASCII dashboard that shows your week at a glance.

Built with Python and Click, it stores everything in a local SQLite database so there's no account or cloud dependency. Tasks can be tagged, filtered, and sorted with intuitive shorthand commands.

The project started as a personal productivity tool and grew into something I use every day. It's open source and contributions are welcome.
